HTML 5

HTML es el lenguaje de marcado estándar para crear páginas web. Define la estructura del contenido utilizando elementos etiquetados, como encabezados, párrafos, enlaces, imágenes y otros tipos de contenido multimedia. Los elementos HTML se representan mediante etiquetas, y el contenido de una página web está organizado y presentado en función de estas etiquetas.

CSS

CSS se utiliza para describir la presentación y el diseño de una página web. Define cómo se muestran los elementos HTML, incluyendo aspectos como colores, fuentes, márgenes, bordes, espaciados, alineaciones y mucho más. Con CSS, puedes crear diseños visualmente atractivos y consistentes a través de todo un sitio web.
